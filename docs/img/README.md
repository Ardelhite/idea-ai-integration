# Screenshots

Drop PNGs here with these names — they're referenced by the root `README.md`
and recommended for the Marketplace listing (see `docs/marketplace-listing.md`):

| File | What to capture |
|---|---|
| `chat.png` | A Fugu response with inline tool cards (run command / file edit / web search). |
| `mcp.png`  | Using an MCP server — the **MCP** dropdown set to **All** and a Playwright (`Browser …`) tool card; the opened page visible. |
| `setup.png`| The **Set up Fugu** dialog (API key + Install Codex + Write provider config). |
| `tabs.png` | Several chat tabs (1, 2, 3) with a spinner on the running tab. |

Tips: use a clean theme, ~1000–1400px wide, and crop to the Karato tool window.
The Marketplace shows the first image prominently, so make `chat.png` the strongest.
